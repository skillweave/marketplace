# skillweave marketplace

Claude Code plugin marketplace for the [SkillWeave](https://github.com/skillweave)
ecosystem.

## Plugins

- [`loom`](https://github.com/skillweave/loom) — universal plugin providing
  skills, agents, first-class team templates, and hooks that adapt to every
  project. Currently pre-alpha (`0.1.0-alpha.1`, scaffold only).

## Install

```
/plugin marketplace add skillweave/marketplace
/plugin install <plugin>@skillweave
```

## Updating

```
/plugin marketplace update skillweave
/plugin update <plugin>
```
